Задача :
Написать программу, которая из имеющегося массива строк формирует массив из строк,
 длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры,
  либо задать на старте выполнения алгоритма. При решение не рекомендуется пользоваться коллекциями,
   лучше обойтись исключительно массивами.
   
Описание алгоритма решения:
Сначало объявляется два массива: изначальный и вторый такой же длины.
 Потом метод, в котором цикл соразмерный длине массива,
  внутри цикла проверка условия ( <=3 ),
   если да элемент первого массива заносится в count элемент
    второго массива. Переменная count чтобы поочередно закидывать
     из первого массива во второй и чтобы потом не было пробелов.
      После присвоения увеличивается переменная count на 1 и возвращается
       к циклу for в котором i увеличивается на 1.
        И так проверяется до конца.
        Потом выводим массив на экран.

2 вариант:
Решение через метод void.
В первом методе void заводятся два строчных массива. Потом метод,
 в котором цикл соразмерный длине массива, внутри которого идет
  проверка условия ( < = 3 ), если да i первого массива заносится
   в i второго. после проверки i увеличивается на 1,
    и так проверяется до конца массива.

Во втором методе void заводится строчный массив. потом метод,
 в котором цикл соразмерный длине массива, выводит сообщение о заполнении
  и заполняется (i+1) элементами (+1- нужен для того,
   чтобы после ввода элемента мы перешли на следующий по возрастанию
    индекс при вводе).после ввода, i увеличивается на 1 и возвращается
     к циклу for. и так до конца массива.

Третий метод void выводит конечный массив на экран.
 для начала заводит строчный массив. после выводит сообщение
  об элементах "равных или меньше 3:", и после методом, в котором цикл
   соразмерный длине массива, выводит i из второго массива,
    в который заносились значения меше 4 элементов. потом, возвращаемся
     к циклу for, в которм i увеличивается на 1 и так далее до конца массива. 

основной код:
отчистка консоли, сообщение о вводе количества элементов массива,
 ввод этих элементов. создаем два массива и присваем значения элементов
  вводимых ранее. 
поочередно вызываем методы void:
(метод заполнения, метод проверки условия, метод вывода конечного массива)
